在前端开发过程中，每次新建一个工程，都要费时费力地把一些js库拷贝到工程目录下，要么就是用相对路径一层层地找，
然而如果自己能有个cdn库，需要的时候只需要引用在线的库文件就好了

这里使用的是阿里云+nodejs的环境，具体的配置见
Node.js笔记（十一）阿里云部署Node.js服务器(Windows环境)

有了这样的环境，只需要简单地几步：
1. npm install -g express
2. 建立server.js
3. node server.js

server.js代码如下：

var express = require('express');
var app = express();
var http = require('http');
var serveStatic = require('serve-static');
app.use(serveStatic(__dirname + '/public'));
var port = 3000;

var httpServer = http.createServer(app);    
httpServer.listen(port, null, function() {
console.log('HTTP server started: http://localhost:' + port);
});

接下来只要把你的库文件扔进public文件夹里就能访问啦
比如说我把一个page.css扔进了public/css下
就可以用

<link href="http://likaiboy.com/css/page.css" rel="stylesheet">

来引用这个文件
